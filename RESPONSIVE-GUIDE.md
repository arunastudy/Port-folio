# 📱 Руководство по адаптивности портфолио

## 🎯 Обзор

Портфолио полностью адаптировано для всех устройств с особым вниманием к популярным моделям iPhone и планшетам.

## 📐 Breakpoints (Контрольные точки)

### Desktop
- **1200px+** - Полноэкранный режим
- **1024px - 1200px** - Малый десктоп / большой планшет

### Tablet
- **768px - 1024px** - Планшеты (iPad, Galaxy Tab)
  - Hero секция вертикально
  - Статистики: 2x2 сетка
  - Проекты: 2 колонки
  - Контакты: 2 колонки

### Mobile
- **480px - 768px** - Большие телефоны
  - Компактное меню (200x300px)
  - Статистики: 2x2
  - Проекты: 1 колонка (max-width: 400px)
  - Книги: горизонтально (160px изображения)
  - Контакты: 2 колонки

- **375px - 480px** - Средние телефоны
  - Книги: 140px изображения
  - Уменьшенные отступы
  - Оптимизированные шрифты

- **360px и меньше** - Маленькие телефоны
  - Минимальные отступы
  - Меню: 170x270px
  - Книги: 110px изображения

## 🍎 Специфичные модели iPhone

### iPhone 14/15/16 Pro Max (430px)
- Hero resume: 320px x 480px
- Заголовок: 2.5rem
- Оптимизировано для больших экранов

### iPhone 12/13/14/15 (390px)
- Hero resume: 290px x 440px
- Статистики: компактные карточки
- Книги: 125px изображения

### iPhone X/XS/11 Pro/12 mini/13 mini (375px)
- Hero resume: 280px x 425px
- Заголовок: 2rem
- Статистики: минимальные отступы
- Книги: 120px изображения

### iPhone XR/11 (414px)
- Hero resume: 300px x 450px
- Балансированные размеры

## 🎨 Мобильное меню

### Структура
```
Header:
├── Logo (arunastudy)
├── Theme Toggle (день/ночь)
├── Language (EN / RU)
└── Burger Menu (справа)
```

### Модальное окно меню
- **Позиция**: Правый верхний угол
- **Размер**: 200px x 300px
- **Стиль**: Современная карточка с blur
- **Анимация**: Плавное появление с задержкой для каждого пункта
- **Закрытие**: 
  - Клик по пункту меню
  - Клик вне меню
  - Клик на оверлей
  - Кнопка ESC

### Особенности
- ✅ Затемнение фона (overlay)
- ✅ Blur эффект
- ✅ Плавные анимации
- ✅ Адаптивные размеры для разных экранов
- ✅ Блокировка скролла при открытом меню

## 📊 Оптимизация элементов

### Hero Section
- **Desktop**: Горизонтальный layout
- **Tablet**: Вертикальный layout, центрирование
- **Mobile**: 
  - Резюме: адаптивный размер
  - Статистики: 2x2 сетка
  - Кнопки: гибкая ширина

### Projects
- **Desktop**: 3 колонки
- **Tablet**: 2 колонки
- **Mobile**: 1 колонка (центрированная, max-width)

### Books
- **Desktop/Tablet**: Горизонтальный layout
- **Mobile**: Горизонтальный, уменьшенные изображения
- **Нумерация**: Адаптивный размер (5rem → 2rem)

### Contact Cards
- **Desktop**: 1 ряд
- **Tablet**: 2 колонки
- **Mobile**: 2 колонки (компактные)

### Experience Timeline
- **All**: Вертикальная линия слева
- **Mobile**: Уменьшенные отступы и маркеры

1. Открыть DevTools (F12)
2. Toggle Device Toolbar (Ctrl+Shift+M)
3. Выбрать устройство или ввести custom размер
4. Тестировать все breakpoints
