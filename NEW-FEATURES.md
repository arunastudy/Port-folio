**Функциональность:**
- Красивый анимированный логотип
- Спиннер загрузки

**Файлы:**

- `src/ts/loader.ts` - логика

- ✅ Вращающийся спиннер
- ✅ Мультиязычность
- ✅ Адаптивность
- ✅ Fallback через 3 секунды

### 2. 📊 Scroll Progress Bar (Индикатор прокрутки)
**Функциональность:**
- Тонкая полоса вверху страницы
- Показывает прогресс прокрутки
- Градиентный цвет
- Свечение для выразительности

**Файлы:**
- `src/styles/scroll-progress.css` - стили
- `src/ts/scroll-progress.ts` - логика
- Добавлен в `index.html`

**Фичи:**
- ✅ Real-time обновление
- ✅ Градиентный дизайн
- ✅ Box-shadow свечение
- ✅ Адаптивная высота (3px → 2px на мобильных)
- ✅ Плавная анимация

---

### 3. ⬆️ Scroll to Top Button (Кнопка "Наверх")
**Функциональность:**
- Круглая кнопка справа внизу
- Появляется после прокрутки 500px
- Плавная прокрутка наверх
- Hover эффекты

**Файлы:**
- `src/styles/scroll-to-top.css` - стили
- `src/ts/scroll-to-top.ts` - логика
- Добавлена в `index.html`

**Фичи:**
- ✅ Автоматическое появление/скрытие
- ✅ Иконка стрелки вверх
- ✅ Hover анимация (поднимается)
- ✅ Box-shadow свечение
- ✅ Touch-friendly размер (44px+ на мобильных)
- ✅ Адаптивные размеры для всех экранов

**Позиция:**
- Desktop: `bottom: 2rem; right: 2rem;` (50×50px)
- Tablet: `bottom: 1.5rem; right: 1.5rem;` (44×44px)
- Mobile: `bottom: 1rem; right: 1rem;` (40×40px)

## 🎨 Кастомизация

### Изменить цвета:
```css
/* В loader.css, scroll-progress.css, scroll-to-top.css */
background: var(--accent-primary); /* Замените на свой цвет */
```

### Изменить таймеры:
```typescript
// В loader.ts
setTimeout(() => { /* ... */ }, 500); // Задержка перед скрытием
```

### Изменить высоту progress bar:
```css
/* В scroll-progress.css */
height: 3px; /* Ваша высота */
```

### Изменить порог появления кнопки:
```typescript
// В scroll-to-top.ts
if (window.scrollY > 500) { /* Ваше значение */ }
```
