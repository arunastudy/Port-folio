# 📱 Исправления для мобильных устройств

## ✅ Выполненные исправления

### 1. 🚫 Убран горизонтальный скролл
**Проблема:** На iPhone появлялся горизонтальный скролл

**Решение:**
- Добавлен `overflow-x: hidden` в html и body
- Добавлен `max-width: 100vw` в body
- Уменьшена ширина проектов с 400px до 340px
- Добавлен `width: 100%` в project-card

**Файлы:**
- `src/styles/base.css`
- `src/styles/responsive.css`

---

### 2. ⚪ Убрана затемненность overlay в меню
**Проблема:** При открытии меню был темный полупрозрачный фон

**Решение:**
- Удален псевдоэлемент `::before` с overlay
- Меню теперь просто белое окно без затемнения фона
- Сохранены все анимации и функциональность

**Файлы:**
- `src/styles/header.css`

**Результат:**
```
Было: [Темный overlay] + Меню
Стало: Меню (просто белое окно)
```

---

### 3. 🎯 Центрирован текст в кнопках Hero
**Проблема:** Текст в кнопках "Мои проекты" и "Связаться" был не по центру

**Решение:**
- Добавлен `justify-content: center` для кнопок
- Добавлен `text-align: center`

**Файлы:**
- `src/styles/responsive.css`

**Применяется к:**
- `.btn-primary` (Мои проекты)
- `.btn-outline` (Связаться)

---

### 4. 📚 Новая структура карточек книг для телефонов
**Проблема:** Карточки были неудобными на маленьких экранах

**Новая структура:**
```
┌─────────────────────────────────┐
│ 01                              │ ← Номер
│ ┌────┐  Дата публикации         │
│ │Фото│  Название книги          │ ← Header (фото + инфо)
│ └────┘                          │
│                                 │
│ Описание книги на полную        │
│ ширину карточки с подробным     │ ← Description
│ текстом о содержании...         │
│                                 │
│ [Tag1] [Tag2] [Tag3] [Tag4]    │ ← Tags
└─────────────────────────────────┘
```

**Изменения:**
1. **Верхняя часть (book-header):**
   - Фото слева
   - Дата и название справа
   - Горизонтальный layout

2. **Нижняя часть (book-content):**
   - Описание на полную ширину
   - Теги на полную ширину

**Размеры фото по экранам:**
- **768px:** 120×160px
- **480px:** 100×140px
- **390px:** 90×130px
- **375px:** 85×120px
- **360px:** 80×110px

**Файлы:**
- `src/styles/responsive.css` - стили
- `src/ts/books.ts` - HTML структура

**Преимущества:**
- ✅ Компактнее
- ✅ Лучше читаемость
- ✅ Логичная структура
- ✅ Описание не обрезается

---

### 5. 👁️ Увеличена видимость Scroll to Top button
**Проблема:** Кнопка была плохо видна на iPhone

**Решение:**
1. **Увеличены размеры:**
   - 768px: 52×52px (было 46×46px)
   - 480px: 50×50px (было 44×44px)
   - 375px: 48×48px (было 42×42px)

2. **Усилен border:**
   - С 2px до 2.5px на мобильных

3. **Добавлены тени:**
   - Normal: `0 4px 16-18px rgba(248, 81, 37, 0.3-0.4)`
   - Visible: `0 6px 20-22px rgba(248, 81, 37, 0.4-0.5)`
   - Более яркое свечение для заметности

4. **Увеличены иконки:**
   - 768px: 24×24px
   - 480px: 22×22px
   - 375px: 20×20px

**Файлы:**
- `src/styles/scroll-to-top.css`

**Результат:**
- ✅ Кнопка хорошо видна
- ✅ Легко нажимается (touch-friendly)
- ✅ Заметное свечение
- ✅ Достаточный размер для всех iPhone
